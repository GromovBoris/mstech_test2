# Test Task for the Position of HTML Layout Designer

## Part 2

Match the block provided on the part_2.mp4 video (available via the link with additional materials) as closely as possible.

### Task requirements:

1. The layout should be fully adaptive (rubbery): all elements should scale smoothly when resizing the browser window, as shown in the video.

2. The video should be played as a whole, not in random pieces.

3. In the layout, all resources must be connected locally.

4. It is not allowed to use libraries, html/css frameworks, or grids like bootstrap, etc.

5. The final project should contain only regular html/css/js files (without preprocessors and assemblers).

6. The completed test task must be uploaded to github with a page on github pages, netlify or vercel.

7. The test task must be sent with a link to the deployment + a link to the github.
